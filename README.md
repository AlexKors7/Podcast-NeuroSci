# "QuironPod: O Podcast sobre Neurociência de e para Mentes Curiosas"
Projeto "QuironPod: O Podcast sobre Neurociência de e para Mentes Curiosas": Esse projeto visa abordar assuntos relacionados à Neurociências (Neuroanatomia, Neurofisiologia e Biofísica) de forma descontraída e atraente para os interessados e curiosos.
